# VostanEditor
## Install prerequisites
```
sudo apt-get install g++ make cmake pkg-config libcurlpp-dev libboost-all-dev
```

## Build
```
mkdir -p cmake/build
cd cmake/build
cmake ../..
make
```

## Usage
```
./VostanEditor
```
